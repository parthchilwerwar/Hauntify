## Development Tools

### Primary Development Environment
- **[Kiro IDE](https://kiro.ai)** - AI-assisted development environment
  - Used for: Full-stack development, code generation, debugging, testing
  - Features utilized: Spec-driven development, AI pair programming, automated workflows

### Design & Prototyping
- **[v0.dev](https://v0.dev)** - Initial UI component generation
  - Used for: Rapid prototyping of React components
  - Components generated: Chat interface, map layout, audio player

### AI & APIs
- **[Groq](https://groq.com)** - LLaMA 3.3 70B Versatile model
  - Used for: Real-time horror story generation with streaming
  - Features: Token-by-token streaming, timeline extraction, atmospheric narratives

- **[ElevenLabs](https://elevenlabs.io)** - Text-to-Speech API
  - Used for: Professional voice narration with multiple voice types
  - Voices: Rachel (Narrator), Antoni (Villain), Dorothy (Ghost), Arnold (Historian)

- **[OpenStreetMap](https://www.openstreetmap.org)** - Map data and tiles
  - Used for: Interactive map visualization
  - Services: Nominatim geocoding, OSM tile server

## Technology Stack

### Frontend Framework
- **Next.js 16** - React framework with App Router
- **React 19** - UI library
- **TypeScript 5** - Type-safe development
- **Tailwind CSS v4** - Utility-first styling

### State & Data Management
- **Zustand** - Lightweight state management
- **Zod** - Runtime validation and type safety
- **localStorage** - Session persistence

### UI Components
- **shadcn/ui** - Pre-built accessible components
- **Radix UI** - Headless UI primitives
- **Lucide React** - Icon library
- **Sonner** - Toast notifications

### Maps & Geolocation
- **Leaflet** - Interactive map library
- **react-leaflet** - React bindings for Leaflet
- **Nominatim** - OpenStreetMap geocoding service

### Audio
- **HTML5 Audio API** - Native audio playback
- **Custom AudioQueueManager** - Queue management and controls
- **Web Speech API** - Fallback TTS
